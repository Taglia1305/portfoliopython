# Data Directory Overview

This sub-directory contains the datasets used for the analysis.

### Content
**Raw Logs**: Original datasets containing timestamped records of calls, durations, and cell identifiers.
**Processed Data**: Output files generated after cleaning and billing calculations.

### Data Privacy & Format
All data is processed in compliance with standard formatting requirements (e.g., CSV or TXT). Ensure that any new datasets added to this folder follow the predefined structure to maintain compatibility with the analysis scripts.
