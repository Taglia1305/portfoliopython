# Telecommunications Traffic Analysis

This project focuses on processing and analyzing large datasets of telephone records to automate billing and monitor network performance.

## Project Overview
The software performs automated data processing on raw traffic logs, applying business logic to calculate costs, apply bonuses, and identify potential network issues such as cell congestion.

### Key Features
* **Automated Billing**: Calculation of total costs based on call duration and specific rate plans.
* **Data Cleaning & Validation**: Implementation of filters to identify and remove corrupted or inconsistent records.
* **Statistical Analysis**: Generation of reports on average call duration and traffic distribution.
* **Network Monitoring**: Identification of high-traffic "cells" to analyze potential network congestion.

### Technical Implementation
* **File I/O**: Efficient management of input/output streams for handling large text-based datasets.
* **Data Structures**: Use of advanced Python structures for fast lookup and data aggregation.
* **Edge Case Management**: Robust error handling for missing values or malformed data strings.
